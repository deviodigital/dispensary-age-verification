== Description ==

Check a visitors age before allowing them to view your dispensary website.

Built and released as an official add-on for [WP Dispensary](https://www.wpdispensary.com)

== Installation ==

1. Upload `dispensary-age-verification` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
